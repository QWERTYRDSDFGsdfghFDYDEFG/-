# When the Sword Arrives

一个基于《剑来》剧情改编的视觉小说项目，
主要用于剧情演出练习，以及完整游戏开发流程的探索。

---

## 📖 项目简介

本项目为视觉小说（Visual Novel），
以《剑来》书简湖篇为核心进行改编。

时间线从 **陈平安离开大隋书院分别之后** 开始，
一直推进到 **崔巉布下书简湖问心局**。

相比传统“爽文式”的强弱对抗，本作更关注一些更慢、更沉的东西：

* 人在规则之中的选择
* 情理与规矩之间的拉扯
* 亲疏远近带来的判断偏差
* 普通人在规则保护下的生存空间
* 成长过程中不可避免的代价

希望通过对话、分支和演出，让这些问题真正“落到玩家手里”。

## 🔄 从 Galgame 转向视觉小说的思考

这个项目最初在介绍时使用了"Galgame"这个说法，但实际上它更接近纯粹的**视觉小说（Visual Novel）**。两者之间存在一些本质上的不同，转变过程中主要有以下几点难度：

1. **叙事结构不同**
   Galgame 的核心是多条角色路线（Route），通常围绕玩家与特定角色的关系发展，分支点明确。
   视觉小说则更侧重于主线故事的叙事质量，分支不一定以"攻略谁"为目标，而是落在道德选择、立场取舍上。
   → 本作放弃了"攻略路线"的设计，转而专注于陈平安的成长轨迹与书简湖的人心考验。

2. **玩家互动逻辑不同**
   Galgame 需要好感度、关系值等数值驱动；视觉小说更依赖场景演出、对话节奏和信息揭示节奏来推动情感。
   → 本作的选项设计不以数值为标准，而是以"立场认同"为核心，让玩家真正面对问题而非管理角色关系。

3. **美术与演出目标不同**
   Galgame 通常需要大量立绘与 CG；视觉小说可以更注重背景氛围与演出节奏。
   → 当前阶段以背景图与文字演出为主，立绘与 CG 为后期补充目标，而非前置需求。

4. **受众预期不同**
   Galgame 玩家通常期待角色互动与多周目路线；视觉小说受众更能接受线性或有限分支的深度叙事。
   → 本作目标读者是熟悉《剑来》原著、对剧情演出有兴趣的读者，而非期待"攻略"的玩家。
