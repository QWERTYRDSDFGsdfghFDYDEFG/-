# When the Sword Arrives

一个基于《剑来》剧情 Galgame 项目，  
用于剧情演出练习与游戏开发流程学习。

---

## 📖 项目简介

本项目为文字冒险游戏（Galgame），  
以《剑来》书简湖篇为核心剧情改编。

时间线从 **陈平安离开大隋书院分别** 开始，  
到 **崔巉为陈平安布下书简湖问心局** 为主线。

本作关注的并非“强者为尊”，  
而是“规矩之内的自由”这一主题，  
通过剧情选择与人物对话展现：

- 人性与神性的冲突  
- 顺序与亲疏  
- 规则对普通人的保护意义  
- 成长必须付出的代价
