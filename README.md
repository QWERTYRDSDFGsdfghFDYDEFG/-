# When the Sword Arrives

一个基于《剑来》剧情改编的视觉小说项目，
主要用于剧情演出练习，以及完整游戏开发流程的探索。

---

## 📖 项目简介

本项目为文字冒险游戏（Galgame），
以《剑来》书简湖篇为核心进行改编。

时间线从 **陈平安离开大隋书院分别之后** 开始，
一直推进到 **崔巉布下书简湖问心局**。

相比传统“爽文式”的强弱对抗，本作更关注一些更慢、更沉的东西：

* 人在规则之中的选择
* 情理与规矩之间的拉扯
* 亲疏远近带来的判断偏差
* 普通人在规则保护下的生存空间
* 成长过程中不可避免的代价

希望通过对话、分支和演出，让这些问题真正“落到玩家手里”。
